1. Introduction

This project implements a comprehensive graph sequence analyzer that determines whether a given sequence of non-negative integers can be realized as the degree sequence of a simple graph. Additionally, it generates all possible non-isomorphic graph realizations for graphic sequences.

Key Features:
  • Determination of graphic sequences using the Havel-Hakimi theorem
  • Generation of all possible non-isomorphic graph realizations
  • Interactive GUI for visualization and analysis
  • Comprehensive test suite with various graph families

2. Theoretical Background

2.1 Graph Sequences

Definition:

A sequence of non-negative integers d = (d₁, d₂, ..., dₙ) is called graphic if there exists a simple graph G with vertex set V(G) = {v₁, v₂, ..., vₙ} such that deg(vᵢ) = dᵢ for i = 1, 2, ..., n.

2.2 The Havel-Hakimi Theorem

Theorem:

A sequence of non-negative integers d = (d₁, d₂, ..., dₙ) in non-increasing order is graphic if and only if the sequence

d' = (d₂-1, d₃-1, ..., dd₁+1-1, dd₁+2, ..., dₙ)

is graphic, where d' is reordered in non-increasing order.

Proof Sketch:

  1. If d is graphic, then d' is obtained by removing a vertex of degree d₁ and its edges
  2. If d' is graphic, we can construct a graph with degree sequence d by adding a new vertex and connecting it to the d₁ vertices with highest degrees

2.3 Fundamental Properties

Necessary Conditions:

  1. The sum of degrees must be even (Handshaking Lemma)
  2. The maximum degree must be less than the number of vertices
  3. The number of vertices of odd degree must be even

3. Algorithms

3.1 Graphic Sequence Verification

Havel-Hakimi Algorithm:

Input: Sequence d = (d₁, d₂, ..., dₙ)
Output: Boolean indicating if d is graphic

Algorithm HavelHakimi(d):
    while d is not empty:
        Sort d in non-increasing order
        if all elements are 0:
            return True
        d₁ = first element of d
        if d₁ ≥ len(d) or d₁ < 0:
            return False
        Remove d₁ from d
        For i from 1 to d₁:
            if i > len(d):
                return False
            d[i] = d[i] - 1
    return True

Time Complexity: O(n²) where n is the length of the sequence

Space Complexity: O(n)
